# **Ruta más corta con Dijkstra (Perú)**

Aplicación web que calcula la ruta más corta entre departamentos del Perú usando el algoritmo de Dijkstra y datos reales de Google Maps.


## **Funcionalidad:**
Selección de origen y destino\
Cálculo de la ruta óptima\
Distancia total y tramos\
Visualización en mapa interactivo


## **Ejecución:**
pip install flask googlemaps polyline\
python app.py

## **Abrir en navegador:**
http://127.0.0.1:5000/\

## **Nota:**
Configura tu API KEY de Google Maps en app.py.
